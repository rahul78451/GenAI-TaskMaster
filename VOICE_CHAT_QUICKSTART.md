## 🐛 Troubleshooting

### "Microphone not working"
1. Check browser has permission (icon next to URL)
2. Ensure microphone is connected
3. Test microphone in system settings
4. Try a different browser

### "No audio output"
1. Check if speaker/headphones are connected
2. Verify system volume is up
3. Check browser volume controls
4. Refresh the page

### "AI not responding"
1. Ensure backend is running (http://localhost:8000)
2. Check internet connection
3. Open browser console (F12) for errors
4. Verify API key is set in backend .env file
